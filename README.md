
# Load testing GUI

## Installation
### Requirements
- tk - `sudo pacman -S tk`
- Docker & Docker-compose latest - `sudo pacman -S docker docker-compose`
### Cloning the Repository

Clone the repository to your local machine:

```bash
git https://github.com/Maclovi/load_testing_gui
cd load_testing_gui
```
## How to run

Just:
```bash
pip3 install .
run
```
